Class omDraggable
  • 预览
  • 属性
  • 方法
  • 事件

用来提供拖动功能.
特点：

  1. 轻量级，简单易用。
  2. 可限制拖动的范围及方向。
  3. 可自定义鼠标在拖动时的样式。
示例：
<script type="text/javascript" >
$(document).ready(function() {
    $('#selector').omDraggable();
});
</script>

<div id="selector">
</div>

omDraggable(p)
参数:
p - 标准config对象：{}

属性

  • axis

    Type:
      String
    Default:
    无

    指定拖动的方向,提供的取值有“x”，“y”，默认不指定方向，可以任意拖动。

    示例：

    //只能沿着x轴的方向拖动
$("#selector").omDraggable({axis:"x"});

  • cancel

    Type:
      Selector
    Default:
    :input,option

    不能启动拖动操作的区域。

    示例：

    //$("#selector")的子元素的<p />不能能够启动拖动操作
$("#selector").omDraggable({cancel:"p"});

  • containment

    Type:
      Selector,Element,String,Array
    Default:
    无

    设置拖动的范围,不能拖动到该范围以外的地方，默认不指定拖动的范围，可以任意拖动。 其值可以是：“parent”、“document”、“window”、[x1,y1,x2,y2]等。

    示例：

    //只能在上一级父元素范围内拖动
$("#selector").omDraggable({containment:"parent"});

  • cursor

    Type:
      String
    Default:
    “auto”

    设置鼠标拖动时的样式。其值为CSS中cursor属性的取值。

    示例：

    //拖动元素时，鼠标呈现十字状
$("#selector").omDraggable({cursor:"crosshair"});

  • disabled

    Type:
      Boolean
    Default:
    false

    是否可拖动。

    示例：

    //元素不可拖动
$("#selector").omDraggable({disabled:true});

  • handle

    Type:
      Selector
    Default:
    无

    能够启动拖动操作的区域，默认不指定区域，拖动元素内的所有区域都可以启动拖动操作。

    示例：

    //只有$("#selector")的子元素的<p />内才能够启动拖动操作
$("#selector").omDraggable({handle:"p"});

  • helper

    Type:
      String,function
    Default:
    "original"

    提供一个辅助的元素作为元素被拖动时的展现，默认为被拖动元素本身作为拖动时的展现元素。

    示例：

    //$("#selector")的clone元素将作为辅助元素
$("#selector").omDraggable({helper:"clone"});

  • revert

    Type:
      Boolean,String
    Default:
    false

    设置当拖动结束后，元素是否会返回到初始位置。默认为false，不会返回；反之为true时，返回到初始位置。 String类型的取值有：“valid”，“invalid”。如果该值为“invalid”，则当元素没有拖动到目的位置时返回；反之为“valid” 时，当元素拖动到目的位置返回。

    示例：

    //元素没有拖动到目的位置时，返回到原位。
$("#selector").omDraggable({revert:"invalid"});

  • scroll

    Type:
      Boolean
    Default:
    true

    拖动元素时，是否自动滚屏。

    示例：

    //拖动元素时，不自动滚屏
$("#selector").omDraggable({scroll:false});

方法

  • destroy

    Signature:
      destroy()

    删除元素的拖动功能.

    返回:
    JQuery对象

    示例：

    var $selector = $("#selector").omDraggable('destroy');

事件

  • onDrag

    function(ui, event)

    拖动时触发事件，当返回为false时，将取消拖动操作。

    参数:
    ui - Objec对象。包括四个属性：helper，position(当前位置)，originalPosition(原始位置)，offset(偏移量)
    event - jQuery.Event对象。

    示例

      $("#selector").omDraggable({onDrag : function(ui, event) {doSomething...}});

  • onStart

    function(ui, event)

    开始拖动时触发事件。

    参数:
    ui - Objec对象。包括四个属性：helper，position(当前位置)，originalPosition(原始位置)，offset(偏移量)
    event - jQuery.Event对象。

    示例

      $("#selector").omDraggable({onStart : function(ui, event) {doSomething...}});

  • onStop

    function(ui, event)

    停止拖动时触发事件。

    参数:
    ui - Objec对象。包括四个属性：helper，position(当前位置)，originalPosition(原始位置)，offset(偏移量)
    event - jQuery.Event对象。

    示例

      $("#selector").omDraggable({onStop : function(ui, event) {doSomething...}});
